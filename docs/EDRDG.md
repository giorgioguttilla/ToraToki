# EDRDG attribution

This app uses data derived from JMdict / JMDict-Simplified.

- JMdict is maintained by the Electronic Dictionary Research and Development Group (EDRDG).
- JMDict-Simplified is a JSON transformation of JMdict provided by the upstream `scriptin/jmdict-simplified` project.
- The packaged app should keep this attribution with any bundled dictionary assets.

Project links:

- https://www.edrdg.org/jmdict/j_jmdict.html
- https://github.com/scriptin/jmdict-simplified
